# 示例代码块

## Python 示例

```python
def greet(name):
    print(f"Hello, {name}!")

greet("Python Developer")
```

## Java 示例

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, Java World!");
    }
}
```
